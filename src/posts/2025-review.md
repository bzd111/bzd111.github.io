#  前言

2025年，变化真的很多。最大的变化是**工作变动**，如果公司不降薪，应该都不会发生这么多变化。


#  工作

公司业务不太好，投资人的钱也不愿意进来。公司从2025年1月就开始降薪，裁员也只给1个月。

工资一般是**拖欠两个月以上**。一月份的工资，三月份发。结果我们组的人全跑了。

面了好多公司，也面了好多人，新来的两个也没呆多久，就跑路了。

能明显感到**市场越来越差**，工资也给不高。

#  学习

说起来，很惭愧，今年学习比较少，还有几篇blog没有发出来。

1. 跟着 [TinTinLand](https://tintinland.com/) 学了一些 **Web3** 的知识。
2. 学了一点AI概念和工作流，speckit、mcp、subagent、agent等

#  生活

由于远程工作的原因，在老家呆了几个月。发现消费是真的低，点个奶茶都要5块钱配送费，难得进一次城。

老婆又回之前的公司，现在又回杭州了，生活便利很多。

这次租房真是踩了很多坑：

1. **采光差**：房间不是正南，阳光不充足
2. **异味**：之前养宠物，房间太臭了，冰箱也臭
3. **虫害**：有蟑螂
4. **设施**：洗衣机漏水


#  26计划

2. **好好学习**
3. **好好锻炼**


# 📝 小结

2025年真是魔幻的一年，开年天崩开局，后面慢慢走上正轨。
